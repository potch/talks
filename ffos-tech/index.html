Kinds Of Firefox OS Apps

"Hosted App"

  • Existing Web Site + App Manifest = Hosted App
  • Performs like a website
  • Easy to build
  • Fewer privileges

"Packaged App"

  • .zip of Static web resources (including App Manifest)
  • Performs more like a traditional app
  • Larger learning curve
  • More privileges

The Manifest

  • A JSON file contaning metadata about your app
  • name, icon, etc.
  • application/x-web-app-manifest+json
{
    "name": "TicTacApp",
    "description": "A simple tic tac toe game",
    "developer": {
        "name": "Potch",
        "url": "http://github.com/potch"
    },
    "launch_path": "/index.html",
    "default_locale": "en",
    "icons": {
        "128": "/icon-128.png"
    }
}

Secret Ingredients

  • Loads content/data asynchronously
